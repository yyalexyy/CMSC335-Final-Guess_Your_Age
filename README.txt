App: Guess Your Name (https://guess-your-age.herokuapp.com/)

Team Members: Alex Juan & Michael Lacanilao 

App Description:
Our project predicts users' age from their name by using the AGIFY API and store name and age into MongoDB.
Users are also able to view all predicted ages on the leaderboard, retrieved from MongoDB.

API link: https://agify.io/

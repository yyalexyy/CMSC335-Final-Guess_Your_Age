App: Guess Your Name (https://guess-your-age.herokuapp.com/)

Team Members: Alex Juan & Michael Lacanilao 

App Description:
Our project predicts users' age from their names by using AGIFY API and stores names and ages into MongoDB.
Users can also view all predicted ages on the leaderboard, retrieved from MongoDB.

API link: https://agify.io/
